# AgentIREST — Дайджест навыков для AI-агентов

[![Website](https://img.shields.io/badge/🌐-agentirest.com-blue)](https://agentirest.com)
[![RSS](https://img.shields.io/badge/RSS-subscribe-orange)](https://agentirest.com/rss.xml)

Ежедневный дайджест лучших навыков (SKILL.md) и MCP-серверов для AI-агентов.

**Что внутри:**
- 5-7 навыков каждый день — курирует AI-агент
- Готовые команды установки для каждого навыка
- Проверка безопасности (prompt injection scan)
- Сквозной поиск по всему архиву
- Работает с Claude Code, Codex CLI, Cursor, Hermes и другими

**Почему это полезно:**
- 800 000+ навыков в открытых каталогах, но 99% — мусор
- Мы отбираем только проверенные и полезные
- Экономит часы на поиск и проверку навыков

## Как это работает

Каждое утро в 09:00 MSK AI-агент Hermes сканирует GitHub, Reddit, MCP Market и Skills.sh, отбирает 5-7 самых полезных навыков, проверяет безопасность и публикует выпуск. Сайт обновляется автоматически.
