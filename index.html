Playwright MCP Server
Microsoft / GitHub · Browser Automation · вып. #1
Самый популярный MCP-сервер 2026 года. 30 000+ звёзд на GitHub. Позволяет агенту управлять браузером: скриншоты, клики, заполнение форм, E2E-тесты. Эталонный MCP-сервер — с него стоит начинать знакомство с протоколом.
ПроверенMCP
npx @anthropic/create-mcp-server playwright
💡 Подключи в mcp.json: {"mcpServers":{"playwright":{"command":"npx","args":["-y","@anthropic/create-mcp-server","playwright"]}}}. Агент сможет открывать сайты, делать скриншоты, кликать, заполнять формы.
13-Agent Security Orchestrator
Reddit r/ClaudeAI / GitHub · Security · вып. #1
Только что вышедший навык для Claude Code: оркестратор 12 специализированных AI-агентов безопасности. Пентест, SAST, DAST, анализ зависимостей — в одном SKILL.md. Идеальный пример того, как навыки эволюционируют от одиночных скриптов к multi-agent системам.

git clone && cp -r skill/ ~/.claude/skills/security-orchestrator
💡 Скопируй папку навыка в ~/.claude/skills/. Агент автоматически подхватит SKILL.md. Вызов: «просканируй проект на уязвимости».

GitHub MCP Server (Official)
GitHub / Anthropic · DevTools · вып. #1
Официальный MCP-сервер GitHub: issues, PR, code search, управление репозиториями прямо из агента. Стабильный, хорошо документированный, с активным сообществом. Базовая инфраструктура для любого coding-агента.
ПроверенMCP
npx -y @anthropic/create-mcp-server github
💡 Подключи в mcp.json: {"mcpServers":{"github":{"command":"npx","args":["-y","@anthropic/create-mcp-server","github"]}}}. Нужен GitHub токен в переменной GITHUB_TOKEN.
E2B MCP — Secure Code Execution Sandbox
E2B / GitHub · Code Execution / Security · вып. #1
Изолированная среда для безопасного выполнения кода агентом. Критически важный компонент после исследований ToxicSkills: вместо запуска непроверенного кода локально — песочница. Рекомендуем к установке всем, кто даёт агенту доступ к shell.
ПроверенMCP
npx -y @anthropic/create-mcp-server e2b
💡 Нужен E2B API ключ (e2b.dev). После подключения агент выполняет код в песочнице, а не локально. Безопасно для непроверенных навыков.
