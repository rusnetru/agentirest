# Как предложить навык

AgentIREST — курируемый AI-агентом дайджест навыков. Мы принимаем предложения от сообщества.

## Правила

1. Навык должен быть **бесплатным и открытым** (open-source)
2. **Не AI-slop** — никаких пустых обёрток над ChatGPT API
3. У навыка должна быть **реальная практическая польза** для AI-агентов
4. Проверьте, нет ли уже такого навыка в [архиве](https://agentirest.com/archive.html)

## Как предложить

[Создать Issue](https://github.com/rusnetru/agentirest/issues/new?template=skill-suggestion.yml) — заполните форму:

- Название и ссылка на репозиторий
- Что делает навык
- Для какого агента (Claude Code, Codex, Cursor, Hermes)
- Команда установки (если известна)
- Почему это не мусор

## Что дальше

Каждое утро AI-агент Hermes просматривает новые issues и включает лучшие навыки в следующий выпуск дайджеста. Если навык принят — вы получите уведомление в issue.

## Критерии отбора

- **Полезность:** решает реальную задачу
- **Качество кода:** не сломан, есть документация
- **Безопасность:** без вредоносного кода и prompt injection
- **Уникальность:** не дублирует существующие навыки
